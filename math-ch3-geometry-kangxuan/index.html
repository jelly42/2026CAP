3-1 證明與推理

🔑 概念 1：什麼是「幾何證明」？

用白話說：

想像你在跟朋友爭論一件事。你不能只說「我覺得是這樣」，你要一步一步拿出「證據」來說服他。幾何證明就是這件事——你有一些「已經確定是對的規則」（像三角形全等、平行線性質），然後你要用這些規則，像堆積木一樣，一步步推到你想要證明的結論。

三個關鍵零件：
  • 已知：題目告訴你的條件（你手上的牌）
  • 求證：題目要你證明的結論（你要到達的終點）
  • 證明：你用規則一步步走過去的路（你打牌的順序）
    • 一個思考小技巧：逆推式思維

    很多時候正著想很難，所以你要「從終點倒著走」。比如要證明兩條線段相等，你先問自己：「什麼情況下兩條線段會相等？」→「在全等三角形裡，對應邊相等。」→「那我怎麼證明兩個三角形全等？」→ 然後去找條件。這就像解迷宮先從出口往回畫。

    🔑 概念 2：常見的幾何推理思路

    要證明「角度相等」，你可以用這些武器：
  • 三角形全等（對應角相等）
  • 三角形相似（對應角相等）
  • 平行線的截角性質（同位角、內錯角、同側內角）
  • 角平分線的定義
  • 三角形內角和 = 180°
  • 等腰三角形的底角相等
  • 圓心角與圓周角的關係
    • 要證明「長度相等」，你可以用：
  • 三角形全等（對應邊相等）
  • 中垂線的性質（中垂線上的點到兩端等距）
    • 卡住的時候，試著「加輔助線」：

    輔助線就像在地圖上自己畫一條捷徑。常見的輔助線有：連接兩點、作高、作角平分線、作平行線、延長線段、作外接圓。

    📋 題目 我還不確定的地方

    什麼時候該選「全等」還是「相似」？

    → 關鍵判斷：如果題目有「等長」的條件，優先想全等；如果有「比例」的條件或「平行線截出的三角形」，優先想相似。

    🔑 概念 3：平行線性質的應用

    用白話說：

    兩條平行線被一條線切過去的時候，會產生一些「角度必定相等」或「角度加起來必定是 180°」的規律。

  • 同位角相等（在同一側、同一位置的角）
  • 內錯角相等（在內側、交錯位置的角）
  • 同側內角互補（在同一側的內角加起來 = 180°）
    • ⚠️ 常見陷阱

    兩個角的兩邊「分別平行」時，這兩個角要嘛相等、要嘛互補（加起來 180°）。很多人只記得相等，忘了互補的情況。

    🔑 概念 4：角平分線與平行線的組合

    用白話說：

    當角平分線碰上平行線，會自動「製造」出等腰三角形。

    為什麼？因為角平分線把一個角切成兩半，然後平行線又製造了內錯角相等，這樣就會有兩個角相等 → 等腰三角形就出現了。

    💡 技巧 實戰技巧

    看到「角平分線 + 平行線」，馬上找等腰三角形，然後用「等腰 → 等邊」來解題。

    🔑 概念 5：三角形全等性質的應用

    全等的五把鑰匙（判別條件）：
    簡稱條件
    SSS三邊對應相等
    SAS兩邊夾一角對應相等
    ASA兩角夾一邊對應相等
    AAS兩角和非夾邊對應相等
    RHS直角三角形的斜邊和一股對應相等
    ⚠️ 注意 注意！SSA 不能用！

    兩邊和一個非夾角相等，不能保證全等。這是最常見的錯誤。

    💡 技巧 實戰技巧

    題目裡看到「共用邊」，那就是一個免費的相等條件。看到「垂直」，就有 90° 的角可以用。

    🔑 概念 6：三角形相似性質的應用

    用白話說：

    相似就是「形狀一樣、大小不同」。就像影印機把一張圖放大或縮小，角度都不變，但邊長按比例放大縮小。

    相似的判別條件：
    簡稱條件備註
    AA兩組角對應相等最常用！
    SAS 相似兩邊成比例且夾角相等
    SSS 相似三邊成比例
    和全等的關係： 全等是相似的「特殊情況」——比例剛好是 1:1。
    📋 題目 我還不確定的地方

    相似比和面積比的關係？

    → 如果兩個三角形的相似比是 a:b，那面積比是 a²:b²。

    🔑 概念 7：代數證明

    用白話說：

    代數證明和幾何證明的「格式」一模一樣（已知、求證、證明），但處理的對象從「圖形」變成「數字的性質」——特別是奇數、偶數、倍數這些東西。

    核心工具——代數表示法：
  • 偶數 = 2m（m 是整數）
  • 奇數 = 2m + 1（m 是整數）
    • 用這兩個表示法，你可以證明很多事：
    運算奇 + 奇奇 + 偶偶 + 偶
    a + b偶奇偶
    a − b偶奇偶
    a × b奇偶偶
    💡 技巧 實戰技巧

    看到「求證某東西是偶數」，就把它湊成 `2 × (某整數)` 的形式。看到「求證某東西是 n 的倍數」，就把它湊成 `n × (某整數)`。

    🔑 概念 8：比大小與不等式證明

    用白話說：

    要證明 A > B，最常見的方法是證明 A − B > 0。

    兩個重要工具：

    1. 平方差公式：a² − b² = (a+b)(a−b)。如果 a > b > 0，那 a+b > 0 且 a−b > 0，所以 a² − b² > 0，得到 a² > b²。

    2. 算幾不等式：兩個正數的算術平均 ≥ 幾何平均，即 (a+b)/2 ≥ √(ab)。

    ⚠️ 我還不確定的地方

    當 a、b 為負數的時候，a > b 不代表 a² > b²（例如 −1 > −2，但 1 < 4）。要特別小心正負號！

    3-2 三角形的外心、內心與重心

    🔑 概念 9：外心（Circumcenter）

    用白話說：

    你有一個三角形，你想在外面畫一個「剛剛好通過三個頂點」的圓。這個圓的圓心，就是外心。

    外心怎麼找？ 三角形三邊的「中垂線」交於一點，那個點就是外心。 為什麼中垂線？ 因為中垂線上的每個點到線段兩端等距。所以 AB 的中垂線上的點到 A 和 B 等距，BC 的中垂線上的點到 B 和 C 等距。交點就同時到 A、B、C 等距 → 可以當圓心。 外心的位置跟三角形的類型有關：
    三角形類型外心位置
    銳角三角形在三角形內部
    直角三角形在斜邊中點
    鈍角三角形在三角形外部
    外心的角度性質：
  • 銳角三角形：`∠BOC = 2∠A`
  • 鈍角三角形（∠A 為鈍角）：`∠BOC = 360° − 2∠A`
    • 直角三角形的特殊好處： 外心在斜邊中點，外接圓半徑 = 斜邊的一半。即 `OA = OB = OC = AB/2`。
    📋 題目 我還不確定的地方

    鈍角三角形的外心在外部，所以 ∠BOC 的公式不一樣。

    記法： 銳角用 2 倍，鈍角用 360° 減去 2 倍。

    🔑 概念 10：內心（Incenter）

    用白話說：

    你有一個三角形，你想在裡面畫一個「剛剛好碰到三條邊」的圓（像一顆球塞進三角形裡面）。這個圓的圓心，就是內心。

    內心怎麼找？ 三角形三個角的「角平分線」交於一點，那個點就是內心。 內心的位置： 永遠在三角形內部。（不像外心會跑到外面去！） 內心的角度性質： `∠BIC = 90° + ½∠A` 內心與面積的關係： 如果內切圓半徑 = r，三角形周長 = s，那三角形面積 = `½ × s × r`。 直角三角形的內切圓半徑： `r = (兩股和 − 斜邊) / 2 = (AC + BC − AB) / 2`
    ⚠️ 注意 常見的錯誤

    把內切圓半徑公式記成「兩股和一斜邊長」而忘記除以 2！一定要除以 2。

    ⚠️ 容易搞混的地方
  • 外心的角度性質是 `∠BOC = 2∠A`
  • 內心的角度性質是 `∠BIC = 90° + ½∠A`

    • 兩個公式長得有點像但完全不同，考試最愛混著考。

    🔑 概念 11：重心（Centroid）

    用白話說：

    把三角形想像成一塊均勻的三角形紙板。用手指頂住一個點，如果紙板可以完美平衡不掉下來，那個點就是重心。

    重心怎麼找？ 三角形三條「中線」（頂點到對邊中點的連線）交於一點，那個點就是重心。 重心的黃金比例（超重要）： 重心把每條中線分成 2:1，靠近頂點的那段是 2，靠近中點的那段是 1。
    AG = (2)/(3) AD , GD = (1)/(3) AD
    重心的面積性質：
  • 重心到三頂點的連線，把三角形切成 三等份 面積
  • `△AGB = △BGC = △CGA = ⅓ △ABC`
  • 三條中線把三角形切成 六等份 面積
  • 每小塊 = `⅙ △ABC`
    • 重心的位置： 永遠在三角形內部。 重心座標公式： 如果三頂點是 A(x₁,y₁)、B(x₂,y₂)、C(x₃,y₃)：
    G = ((x₁+x₂+x₃)/(3), (y₁+y₂+y₃)/(3))

    🔴 提示問題📝 筆記內容

    證明「角度相等」有哪些方法？① 三角形全等（對應角）② 三角形相似（對應角）③ 平行線截角（同位角、內錯角）④ 角平分線定義 ⑤ 內角和 180° ⑥ 等腰底角相等 ⑦ 圓心角與圓周角
    證明「長度相等」有哪些方法？① 三角形全等（對應邊）② 中垂線性質（中垂線上的點到兩端等距）
    什麼是輔助線？常見有哪些？為了證明方便而自己加畫的線條。常見：兩點連線、作高、作角平分線、作平行線、延長線段、作外接圓

    📐 範例題：SAS 全等證明

    📋 題目 題目

    已知 AB = AC，∠ BAD = ∠ CAD，求證 AD 平分 BC。

    ✅ 詳解 詳解
    【證明】 在 △ ABD 和 △ ACD 中：

    | 條件 | 理由 |

    |:---|:---|

    | AB = AC | 已知 |

    | ∠ BAD = ∠ CAD | 已知 |

    | AD = AD | 共用邊 |


    ⇒ △ ABD ≅ △ ACD（SAS 全等）


    ⇒ BD = CD（對應邊相等）


    故 AD 平分 BC。 ∎

    📌 摘要 摘要

    幾何證明 = 用已知條件 + 定理規則，逐步推到結論。核心策略是逆推式思維：從結論倒推需要什麼條件。

    3-1-② 平行線性質

    🔴 提示問題📝 筆記內容
    平行線被截線切過時，產生哪三種角的關係？① 同位角相等（同側同位置）② 內錯角相等（內側交錯）③ 同側內角互補（同側內角和 = 180°）
    「兩邊分別平行」的兩角有什麼關係？兩種可能：相等或互補（和 = 180°）。不能只記相等！
    角平分線 + 平行線 = ？自動產生等腰三角形。原理：角平分線切出等角 + 平行線的內錯角相等 → 三角形有兩角相等 → 等腰

    📐 範例題：∠A、∠B 兩邊分別平行且互相垂直

    📋 題目 題目

    如圖，∠A、∠B 的一邊互相平行，另一邊互相垂直。求證 ∠ A + ∠ B = 90°。

    ✅ 詳解 詳解
    【思路】 利用內錯角相等 + 直角 = 90°。

    【證明】

    1. 因為兩邊分別平行，截線穿過產生內錯角

    2. 設內錯角為 ∠ A' = ∠ A（內錯角相等）

    3. 在直角 F 處：∠ A' + ∠ B = 90°（兩角互餘，因為另一邊互相垂直）

    4. 所以 ∠ A + ∠ B = 90°。 ∎

    📐 範例題：角平分線與平行線（求比例）

    📋 題目 題目

    平行四邊形 ABCD 中，∠ ABC 的角平分線交 AD 於 E 點。AB = 10，BC = 16。求證 AE : ED = 5 : 3。

    ✅ 詳解 詳解
    【思路】 角平分線 + 平行線 → 等腰三角形 → 求長度。

    【解法】

    1. BE 是 ∠ ABC 的角平分線，所以 ∠ ABE = ∠ EBC

    2. 因為 AD ∥ BC（平行四邊形），所以 ∠ AEB = ∠ EBC（內錯角）

    3. 因此 ∠ ABE = ∠ AEB → △ ABE 為等腰三角形 → AE = AB = 10

    4. 又 AD = BC = 16（平行四邊形對邊相等）

    5. ED = AD - AE = 16 - 10 = 6

    6. AE : ED = 10 : 6 = 5 : 3。 ∎

    📌 摘要 摘要

    平行線截角有三種關係（同位角相等、內錯角相等、同側內角互補）。角平分線 + 平行線必出等腰三角形。

    3-1-③ 三角形全等

    🔴 提示問題📝 筆記內容
    全等的五個判別條件？SSS（三邊）、SAS（兩邊夾角）、ASA（兩角夾邊）、AAS（兩角 + 非夾邊）、RHS（直角 + 斜邊 + 一股）
    SSA 為什麼不能用？兩邊和非夾角相等時，可以構造出兩個不同的三角形（鞦韆效應），所以無法唯一確定三角形
    題目中常見的「隱藏條件」有哪些？① 共用邊（免費的等邊）② 對頂角（免費的等角）③ 垂直（免費的 90°）④ 中點（免費的等長線段）

    📐 範例題：等腰三角形兩腰上的高

    📋 題目 題目

    在 △ ABC 中，AB = AC，BD、CE 分別為兩腰 AC、AB 上的高。若 BE = 5，BD = 12，求 BC。

    ✅ 詳解 詳解
    【思路】 等腰三角形 → 兩腰高相等 → 用全等證 BD = CE，然後用畢氏定理。

    【解法】

    1. 因為 AB = AC（等腰），∠ A 為共用角

    2. 在 △ ABD 和 △ ACE 中：AB = AC，∠ A = ∠ A，∠ BDA = ∠ CEA = 90°

    3. ⇒ △ ABD ≅ △ ACE（AAS 全等）

    4. ⇒ BD = CE（對應邊相等）

    5. 在直角 △ BDC 中（∠ BDC = 90°）：

    BC = √BD² + DC²

    但我們需要另一個角度——

    6. 在直角 △ BCE 中（∠ BEC = 90°）：

    BC = √BE² + CE² = √5² + 12² = √25 + 144 = √169 = 13


    ∴ BC = 13。 ∎

    📌 摘要 摘要

    全等有五把鑰匙（SSS、SAS、ASA、AAS、RHS），SSA 不能用。做題時先找隱藏條件（共用邊、對頂角、垂直）。

    3-1-④ 三角形相似

    🔴 提示問題📝 筆記內容
    相似和全等有什麼差別？全等 = 形狀大小都一樣（比例 1:1）。相似 = 形狀一樣、大小可以不同（比例 a:b）
    相似的判別條件？① AA（兩角相等，最常用）② SAS 相似（兩邊成比例 + 夾角相等）③ SSS 相似（三邊成比例）
    相似比、面積比的關係？相似比 = k:1 → 對應邊比 = k:1 → 面積比 = k²:1
    什麼時候用全等？什麼時候用相似？有「等長」→ 想全等。有「比例」或「平行線截三角形」→ 想相似
    平行線截比例線段是什麼意思？若 DE ∥ BC，則 (AD)/(DB) = (AE)/(EC)。平行線會把三角形的兩邊按同一比例分割

    📐 範例題：直角三角形斜邊上的高（相似性質）

    📋 題目 題目

    △ ABC 中，∠ BAC = 90°，AD 為斜邊 BC 上的高。求證 AD² = BD × CD。

    ✅ 詳解 詳解
    【思路】 利用兩組相似三角形。

    【證明】

    1. 在 △ ABD 和 △ CAD 中：

  • ∠ ADB = ∠ ADC = 90°（AD 為高）
  • ∠ ABD = 90° - ∠ B，∠ ACD = 90° - ∠ C
  • 而 ∠ BAD = 90° - ∠ B = ∠ ACD

    • 2. 所以 △ ABD ∼ △ CAD（AA 相似）

    3. 對應邊成比例：

    (AD)/(CD) = (BD)/(AD)

    4. 交叉相乘：

    AD² = BD × CD


    ∎

    📐 範例題：直角三角形斜邊上的高（計算）

    📋 題目 題目

    △ ABC 中，∠ BAC = 90°，AD ⊥ BC 於 D，AD = 4，BD 比 CD 長 6。求 CD。

    ✅ 詳解 詳解
    【解法】 利用上題結論 AD² = BD × CD。

    1. 設 CD = x，則 BD = x + 6

    2. 代入公式：

    4² = (x+6) × x

    16 = x² + 6x

    x² + 6x - 16 = 0

    3. 因式分解：(x+8)(x-2) = 0

    4. x = 2 或 x = -8（負值不合）


    ∴ CD = 2。 ∎

    📌 摘要 摘要

    相似 = 等比例放大縮小。最常用 AA 判別。相似比 k → 面積比 k²。看到平行線截三角形就想相似。直角三角形斜邊上的高：AD² = BD × CD。

    3-1-⑤ 代數證明

    🔴 提示問題📝 筆記內容
    奇偶數的代數表示法？偶數 = 2m，奇數 = 2m+1（m 為整數）
    奇偶數的加減乘規律？奇 + 奇 = 偶、奇 + 偶 = 奇、偶 + 偶 = 偶。奇 × 奇 = 奇、奇 × 偶 = 偶、偶 × 偶 = 偶。口訣：加減看「奇數的個數」（奇數個奇數 → 奇；偶數個奇數 → 偶），乘法只要有一個偶數結果就是偶數
    怎麼證明某數是 n 的倍數？把它改寫成 n × (某整數) 的形式
    代數證明常用的公式？① a² - b² = (a+b)(a-b) ② (a+b)² = a²+2ab+b² ③ (a-b)² = a²-2ab+b² ④ 畢氏定理 a²+b²=c²
    「餘數」相關的代數表示？a 除以 n 餘 r → 設 a = nk + r（k 為非負整數，0 ≤ r < n）

    📐 範例題：畢氏定理 + 平方差

    📋 題目 題目

    直角三角形三邊長為 6、b、c（b、c 為正整數），其中 c 為斜邊長。求證 c + b 為 36 的因數。

    ✅ 詳解 詳解
    【解法】 利用畢氏定理 + 平方差公式。

    1. 由畢氏定理：c² = 6² + b² = 36 + b²

    2. 移項：c² - b² = 36

    3. 平方差公式：(c+b)(c-b) = 36

    4. 因為 c > b > 0，所以 c+b > 0 且 c-b > 0

    5. (c+b) 和 (c-b) 都是正整數，且乘積 = 36

    6. 所以 c+b 是 36 的因數。 ∎


    【核心技巧】 看到「斜邊 - 某邊」或「斜邊 + 某邊」→ 馬上想到畢氏定理移項成平方差 (c+b)(c-b)。

    📐 範例題：餘數問題

    📋 題目 題目

    若 b 為正整數，且 b 除以 6 後餘 2，則 b 是偶數還是奇數？

    ✅ 詳解 詳解

    1. 設 b = 6k + 2（k 為非負整數）

    2. b = 6k + 2 = 2(3k + 1)

    3. 因為 3k+1 為整數，所以 b 是 2 的倍數


    ∴ b 為偶數。 ∎


    【核心技巧】 設餘數表示式後，嘗試提取公因數 2 來判斷奇偶。
    📌 摘要 摘要

    代數證明的核心：把數字用 2m（偶）或 2m+1（奇）表示，再用公式展開化簡，湊出目標形式。

    3-1-⑥ 比大小與不等式

    🔴 提示問題📝 筆記內容
    證明 a > b 最標準的方法？證明 a - b > 0
    a > b > 0 時，a² 和 b² 的大小？a² > b²。方法：a² - b² = (a+b)(a-b)，因為 a+b>0 且 a-b>0，所以乘積 >0
    如果 a、b 有負數呢？不能直接比平方！ 例如 -1 > -2 但 (-1)² = 1 < 4 = (-2)²。必須先確認正負
    算幾不等式？兩正數 a, b：(a+b)/(2) ≥ √ab，等號成立條件：a = b

    📐 範例題：證明 a² > b²（做差法）

    📋 題目 題目

    已知 a > b > 0，求證 a² > b²。

    ✅ 詳解 詳解
    【證明 1：做差法 + 平方差】

    1. a² - b² = (a+b)(a-b) ……①

    2. 因為 a > b > 0：

  • a + b > 0（兩正數之和）
  • a - b > 0（a > b）

    • 3. 所以在 ① 中，(a+b) 和 (a-b) 都是正數

    4. 正數 × 正數 = 正數

    5. ∴ a² - b² > 0，即 a² > b²。 ∎


    【證明 2：等量乘法】

    1. ∵ a > b > 0，∴ a > a · (b)/(a)... 不對，更直接地：

    2. ∵ a > b > 0 且 a > 0，∴ a · a > a · b，得 a² > ab ……①

    3. ∵ a > b > 0 且 b > 0，∴ a · b > b · b，得 ab > b² ……②

    4. 由 ①、② 得 a² > ab > b²，故 a² > b²。 ∎

    📌 摘要 摘要

    比大小 → 做差法（a-b 是否 >0）。正數可直接比平方，有負數要小心。算幾不等式：算術平均 ≥ 幾何平均。

    3-2-① 三角形的外心

    🔴 提示問題📝 筆記內容
    外心的定義？三角形三邊中垂線的交點
    外心到三頂點的距離？等距：OA = OB = OC（= 外接圓半徑 R）
    為什麼是中垂線？中垂線上任一點到線段兩端等距 → 交點同時到三頂點等距 → 可作為外接圓圓心
    外心的位置？銳角 → 三角形內部。直角 → 斜邊中點。鈍角 → 三角形外部
    外心的角度性質？銳角：∠ BOC = 2∠ A。鈍角（∠A 為鈍角）：∠ BOC = 360° - 2∠ A
    直角三角形外接圓半徑？R = (1)/(2) × 斜邊長。外心就在斜邊中點

    📐 範例題：外心角度計算

    📋 題目 題目

    在銳角 △ ABC 中，O 為外心，∠ BOC = 140°，求 ∠ A 的度數。

    ✅ 詳解 詳解
    【解法】 直接套用外心角度公式。

    1. 銳角三角形的外心角度性質：∠ BOC = 2∠ A

    2. 代入：140° = 2∠ A

    3. ∠ A = 70°


    ∴ ∠ A = 70°。 ∎

    📐 範例題：直角三角形外接圓半徑

    📋 題目 題目

    直角 △ ABC 中，∠ A = 90°，AB = 7，AC = 24，求 △ ABC 的外接圓半徑。

    ✅ 詳解 詳解

    1. 先求斜邊 BC：

    BC = √7² + 24² = √49 + 576 = √625 = 25

    2. 直角三角形的外心在斜邊中點，外接圓半徑 = 斜邊的一半：

    R = (BC)/(2) = (25)/(2) = 12.5


    ∴ 外接圓半徑為 12.5。 ∎

    📌 摘要 摘要

    外心 = 中垂線交點 = 外接圓圓心，到三頂點等距。位置隨三角形類型變化（銳在內、直在斜邊中點、鈍在外）。角度公式：銳角用 2∠ A，鈍角用 360°-2∠ A。

    3-2-② 三角形的內心

    🔴 提示問題📝 筆記內容
    內心的定義？三角形三個角的角平分線交點
    內心到三邊的距離？等距：ID = IE = IF（= 內切圓半徑 r）
    內心的位置？永遠在三角形內部（不像外心會跑到外面）
    內心的角度性質？∠ BIC = 90° + (1)/(2)∠ A
    內心與面積的公式？三角形面積 = (1)/(2) × s × r，其中 s = 周長，r = 內切圓半徑
    面積分割比？△ AIB : △ BIC : △ CIA = AB : BC : CA（底邊比 = 面積比，因為等高 = r）
    直角三角形內切圓半徑？r = (AC + BC - AB)/(2)（兩股和 − 斜邊，再除以 2）

    📐 範例題：內心角度計算

    📋 題目 題目

    I 為 △ ABC 的內心，若 ∠ BIC = 130°，求 ∠ BAC。

    ✅ 詳解 詳解

    1. 內心角度性質：∠ BIC = 90° + (1)/(2)∠ A

    2. 代入：130° = 90° + (1)/(2)∠ A

    3. (1)/(2)∠ A = 40°

    4. ∠ A = 80°


    ∴ ∠ BAC = 80°。 ∎

    📐 範例題：直角三角形內切圓半徑

    📋 題目 題目

    直角 △ ABC，∠ C = 90°，AC = 3，AB = 5，求鋪色部分（內切圓）的面積。

    ✅ 詳解 詳解

    1. 先求 BC：BC = √5² - 3² = √16 = 4

    2. 直角三角形內切圓半徑公式：

    r = (AC + BC - AB)/(2) = (3 + 4 - 5)/(2) = (2)/(2) = 1

    3. 內切圓面積：

    S = π r² = π × 1² = π


    ∴ 鋪色部分面積 = π。 ∎


    【驗證】 用面積公式驗算：
  • △ ABC 面積 = (1)/(2) × 3 × 4 = 6
  • (1)/(2) × s × r = (1)/(2) × (3+4+5) × 1 = 6 ✅
    • ⚠️ 注意 易錯提醒

    ① 公式忘記除以 2 → 半徑算成兩倍！

    ② 和外心公式搞混：外心 ∠ BOC = 2∠ A，內心 ∠ BIC = 90°+(1)/(2)∠ A，完全不同！

    📌 摘要 摘要

    內心 = 角平分線交點 = 內切圓圓心，到三邊等距，永遠在內部。角度：∠ BIC = 90°+(1)/(2)∠ A。面積 = (1)/(2)sr。直角三角形 r = (兩股和 - 斜邊)/(2)。

    3-2-③ 三角形的重心

    🔴 提示問題📝 筆記內容
    重心的定義？三角形三條中線（頂點到對邊中點的連線）的交點
    重心分割中線的比例？2:1（靠頂點端 : 靠中點端）。AG = (2)/(3)AD，GD = (1)/(3)AD
    重心的位置？永遠在三角形內部
    重心的面積性質（三等份）？重心與三頂點連線 → 切成三等份：△ AGB = △ BGC = △ CGA = (1)/(3)△ ABC
    重心的面積性質（六等份）？三條中線 → 切成六等份：每小塊 = (1)/(6)△ ABC
    重心座標公式？G = ((x₁+x₂+x₃)/(3), (y₁+y₂+y₃)/(3))（三頂點座標各取平均）

    📐 範例題：重心與長度

    📋 題目 題目

    △ ABC 的三中線 AD、BE、CF 相交於 G 點。若 AG + BG + CG = 16，求 AD + BE + CF。

    ✅ 詳解 詳解
    【思路】 重心分割中線為 2:1。

    1. 重心性質：AG = (2)/(3)AD，BG = (2)/(3)BE，CG = (2)/(3)CF

    2. 所以：

    AG + BG + CG = (2)/(3)(AD + BE + CF)

    3. 代入已知：

    16 = (2)/(3)(AD + BE + CF)

    4. 解出：

    AD + BE + CF = 16 × (3)/(2) = 24


    ∴ AD + BE + CF = 24。 ∎

    📐 範例題：重心與面積

    📋 題目 題目

    △ ABC 中，三中線交於 G 點。若 △ ABC 面積 = 54，求 △ ABG、△ BDG 及四邊形 CDGE 的面積。

    ✅ 詳解 詳解

    1. 三中線把 △ ABC 切成六等份：

    每小塊 = (54)/(6) = 9

    2. △ ABG = 三等份之一（重心到頂點連線切成三等份）：

    △ ABG = (54)/(3) = 18

    3. △ BDG = 六等份之一：

    △ BDG = 9

    4. 四邊形 CDGE = 兩個小三角形：

    CDGE = 9 + 9 = 18


    ∴ △ ABG = 18，△ BDG = 9，四邊形 CDGE = 18。 ∎

    📌 摘要 摘要

    重心 = 中線交點，分中線為 2:1，永遠在內部。面積切成 3 等份（連頂點）或 6 等份（三中線）。座標 = 三頂點平均。

    ✅ 詳解 詳解
