焦點 1：化學計量

1-1 原子量與分子量

白話解釋

想像你有一堆不同大小的彈珠。我們需要一個「標準彈珠」來比較大小。科學家選了**碳-12**這顆原子當標準，規定它的質量＝12。

>

- **原子量**：就是拿某個原子跟碳-12 比較的「比值」。比如氧原子量是 16，意思是氧比碳-12 重一點點（16/12 倍）。

- **分子量**：就是一個分子裡所有原子量「加起來」。像水 H₂O = 1×2 + 16 = 18。

重要公式：

公式說明
原子量 = 待測原子的質量 / (碳-12 質量 ÷ 12)原子量的定義
分子量 = 各原子量 × 個數之總和分子量的計算

計算範例：

  • H₂O 的分子量 = 1×2 + 16×1 = **18**
  • CO₂ 的分子量 = 12×1 + 16×2 = **44**
  • CH₄ 的分子量 = 12×1 + 1×4 = **16**
  • H₂SO₄ 的分子量 = 1×2 + 32×1 + 16×4 = **98**
    • ⚠️ 我還不確定的地方

    原子量是「比較值」，**沒有單位**！它不是原子的實際質量（那個超級超級小）。容易跟莫耳質量搞混。

    1-2 莫耳（mole）

    白話解釋

    買雞蛋我們說「一打 = 12 顆」，科學家數原子也需要一個「打」。這個「打」就叫**莫耳**，1 莫耳 = 6×10²³ 個（這個數叫亞佛加厥數）。

    >

    為什麼是 6×10²³？因為科學家規定：**12 公克的碳-12 剛好含有 6×10²³ 個碳原子**。

    三大轉換公式：

    轉換方向公式
    質量 → 莫耳數莫耳數 = 物質質量(g) ÷ 分子量(或原子量)
    莫耳數 → 粒子數粒子數 = 莫耳數 × 6×10²³
    莫耳數 → 質量質量(g) = 莫耳數 × 分子量

    關鍵關係圖：

    
質量(g)  ←──÷分子量──→  莫耳數  ←──×6×10²³──→  粒子個數

    計算範例：

  • 20 克 H₂ → 莫耳數 = 20/2 = 10 mol
  • 88 克 CO₂ → 莫耳數 = 88/44 = 2 mol
  • 90 克 H₂O → 莫耳數 = 90/18 = 5 mol
    • ⚠️ 我還不確定的地方

    莫耳是「計量單位」，可以有小數（如 1.6 莫耳），但它數的對象必須是**整數**（不可能有 1.6 個原子）。莫耳數可以是小數，粒子數一定是整數！

    1-3 莫耳質量與質量比

    白話解釋

    - **1 莫耳的質量** = 原子量或分子量，但單位變成「公克」！

    - 例：1 莫耳 H₂O = 18 公克，1 莫耳 CO₂ = 44 公克

    - **1 個原子（分子）的真正質量** = 分子量 ÷ 6×10²³（超級小的數字！）

    - **莫耳數比 = 質量比 ÷ 分子量比**

    範例：已知質量比求莫耳數比

  • 某化合物中 X 和 O 的質量比為 9:8，原子量 X=27, O=16
  • 莫耳數比 = (9/27) : (8/16) = 1/3 : 1/2 = 2:3
  • 所以化合物可能是 X₂O₃（例如 Al₂O₃）

    • 焦點 2：化學反應

    2-1 化學反應常見的現象

    白話解釋

    化學反應就像「變魔術」，原本的東西變成完全不同的新東西。你可以從這些「線索」看出化學反應發生了：

    現象例子反應式
    發光、發熱鎂在氧氣中燃燒，發出白色強光2Mg + O₂ → 2MgO
    顏色改變藍色硫酸銅加熱，變白色CuSO₄·5H₂O → CuSO₄ + 5H₂O
    產生沉澱碳酸鈉 + 氯化鈣 → 白色碳酸鈣沉澱Na₂CO₃ + CaCl₂ → CaCO₃↓ + 2NaCl
    產生氣體碳酸鈣 + 鹽酸 → 冒泡泡CaCO₃ + 2HCl → CaCl₂ + CO₂↑ + H₂O

    2-2 化學反應式的寫法

    白話解釋

    化學反應式就像一個「食譜」：

    - **左邊** = 材料（反應物）

    - **右邊** = 成品（生成物）

    - **箭號 →** = 「變成」

    - **+** = 「加上」

    - **係數** = 告訴你要用幾份

    書寫步驟：

    1. 先寫出反應物和生成物的化學式

    2. 用箭號「→」連接（左邊反應物，右邊生成物）

    3. 平衡係數（讓左右兩邊原子數相等）

    平衡技巧：

  • 先平衡最複雜的分子
  • 如果出現分數，全部乘以最小公倍數變整數

    • 範例：氫氣燃燒生成水

    
未平衡：H₂ + O₂ → H₂O
平衡後：2H₂ + O₂ → 2H₂O
  • 驗證：左邊 H=4, O=2；右邊 H=4, O=2 ✓

    以 2H₂O₂ →(MnO₂) 2H₂O + O₂ 為例：

    係數比 = 莫耳數比 = 分子數比，但**不等於**質量比！質量比要乘上分子量。很多人搞混這個！

    2-4 質量守恆定律

    白話解釋

    想像你把樂高積木從一個造型拆開，重新組成另一個造型。積木的**總重量不變**！化學反應也一樣：

    >

    **反應前所有物質的總質量 ＝ 反應後所有物質的總質量**

    用道耳頓原子說解釋：

  • 化學反應只是原子的「重新排列組合」
  • 原子不會憑空產生或消失
  • 所以總質量不變
    • ⚠️ 我還不確定的地方

    如果反應有氣體跑掉（如 CO₂↑），秤重時質量「看起來」會減少，但實際上加上跑掉的氣體，總質量還是相等的！

    2-5 化學反應式的狀態標示

    白話解釋

    我們可以在化學式旁邊加上「標籤」，告訴別人這東西是什麼狀態：

    - **(s)** = 固體 solid

    - **(l)** = 液體 liquid

    - **(g)** = 氣體 gas

    - **(aq)** = 水溶液 aqueous

    - **↑** = 有氣體產生

    - **↓** = 有沉澱產生

    - **△** = 需要加熱

    - **MnO₂** 寫在箭號上方 = 催化劑

    焦點 3：氧化

    3-1 氧化與活性大小

    白話解釋

    **氧化** = 物質跟氧「交朋友」（結合）。

    - **劇烈的氧化** = 燃燒（一下子就反應完）

    - **溫和的氧化** = 生鏽、呼吸作用、光合作用（慢慢來）

    >

    **活性** = 一個元素有多「想」跟氧交朋友。活性越大，越容易燃燒。

    活性大小排列（口訣：鉀鈉鋇鈣鎂鋁碳鋅鉻鐵錫鉛氫銅汞銀鉑金）：

    
K > Na > Ba > Ca > Mg > Al > C > Zn > Cr > Fe > Sn > Pb > H > Cu > Hg > Ag > Pt > Au
鉀   鈉   鋇   鈣   鎂   鋁   碳  鋅   鉻   鐵  錫   鉛   氫  銅   汞   銀   鉑   金
    口訣

    「**嫁**（鉀）**那**（鈉）**美**（鎂）**女**（鋁），**心**（鋅）**鐵**（鐵）**真**（錫）**前**（鉛），**統**（銅）**共**（汞）**一**（銀）**百**（鉑）**斤**（金）」

    活性元素特性氧化物特性例子
    活性大不安定很安定鎂容易燃燒，氧化鎂安定
    活性小不安定不安定銅不容易燃燒，氧化銅不安定

    3-2 金屬燃燒與燃燒產物

    白話解釋

    不同金屬燃燒有不同的「表演」：

    鈉、鎂、鋅、銅的燃燒比較：

    金屬燃燒難易度火焰顏色燃燒產物氧化物溶水水溶液性質
    鈉 Na容易黃色氧化鈉 NaOH溶鹼性
    鎂 Mg容易白光氧化鎂 Mg(OH)₂難溶鹼性
    鋅 Zn不容易黃綠色氧化鋅難溶-
    銅 Cu不容易-氧化銅（黑色）不溶中性

    非金屬（硫、磷、碳）燃燒比較：

    非金屬火焰產物氧化物溶水水溶液性質
    硫 S藍紫色SO₂ → H₂SO₃（亞硫酸）溶酸性
    紅磷 P黃白色P₂O₅ → H₃PO₄（磷酸）溶弱酸性
    碳 C橘紅色CO₂ → H₂CO₃（碳酸）溶弱酸性
    ⚠️ 我還不確定的地方

    金屬氧化物溶水 → **鹼性**；非金屬氧化物溶水 → **酸性**。但不是所有金屬氧化物都能溶於水！（例如氧化銅不溶）

    3-3 常見鐵製品與防鏽

    白話解釋

    - **白鐵**（鍍鋅）：鋅的活性比鐵大，會先犧牲自己被氧化，保護鐵。即使刮傷，鋅也會先被腐蝕。

    - **亞鉛（馬口鐵）**（鍍錫）：錫的活性比鐵小。如果表面被刮傷，鐵會先被腐蝕，錫沒有保護作用。

    - **鋁的防鏽**：鋁表面會生成緻密的氧化鋁，像一層「保護膜」，阻止內部繼續氧化。

    焦點 4：氧化還原與金屬冶煉

    4-1 氧化還原反應

    白話解釋

    氧化和還原就像「搶氧大作戰」：

    - **氧化** = 得到氧（跟氧結合）

    - **還原** = 失去氧（氧被搶走）

    - 兩者**一定同時發生**！有人被氧化，就一定有人被還原。

    通式：

    
活性大的元素 + 活性小元素的氧化物 → 活性大元素的氧化物 + 活性小的元素

    範例： 2Mg + CO₂ → 2MgO + C

  • Mg 被**氧化**（得到氧，變成 MgO）
  • CO₂ 被**還原**（失去氧，變成 C）

    • 4-2 金屬冶煉（高爐煉鐵）

    白話解釋

    自然界的鐵大多以**氧化鐵（Fe₂O₃）**形式存在（就是鐵礦石）。要把鐵「救」出來，就要用活性更大的東西把氧搶走。

    >

    高爐裡用**煤焦（碳）**當搶氧的角色：

    高爐反應步驟：

    1. 煤焦燃燒產生 CO 和大量的熱：2C + O₂ → 2CO + 熱量

    2. 煤焦跟二氧化碳還原：C + CO₂ → 2CO

    3. 關鍵反應：CO 把鐵礦石中的氧搶走：Fe₂O₃ + 3CO → 2Fe + 3CO₂

    4. 加入石灰石（CaCO₃）去除雜質

    ⚠️ 我還不確定的地方

    高爐煉鐵中，煤焦的用途有**兩個**：①當燃料（提供熱量）②當還原劑（搶氧）。很多題目會問這個！

    4-3 日常生活中的氧化還原

    光合作用 vs 呼吸作用：

    光合作用呼吸作用
    反應式6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O
    CO₂ 的角色被**還原**是**氧化**的產物
    O₂ 的角色是**還原**的產物是**氧化**劑

    生活中的氧化劑與還原劑：

    本身被還原（氧化劑）本身被氧化（還原劑）
    種類氧氣、過氧化氫、次氯酸鈉碳、煤焦、二氧化硫
    應用漂白劑（漂白水 NaClO）、雙氧水煉鐵、食品防腐

    提示問題筆記內容

    📊 摘要： 透過質量比 = 原子量×個數之比，可以反推未知元素的原子量和化學式。

    3-1 活性大小與氧化

    提示問題筆記內容
    活性的定義？依元素在空氣中燃燒的難易程度，比較元素對氧的活性大小
    活性大的特徵？元素本身不安定，容易燃燒；氧化物很安定

    📊 摘要： 活性大→易燃燒→氧化物安定。活性小→不易燃燒→氧化物不安定。活性排列是重要考點。

    3-2 金屬與非金屬的燃燒

    提示問題筆記內容
    金屬氧化物水溶液？多呈**鹼性**（但不是全部都能溶於水）
    非金屬氧化物水溶液？多呈**酸性**
    鈉的燃燒？黃色火焰，產物 Na₂O → 溶水呈鹼性。4Na+O₂→2Na₂O

    📊 摘要： 金屬氧化物溶水呈鹼性，非金屬氧化物溶水呈酸性。燃燒火焰顏色和產物性質是常考重點。
